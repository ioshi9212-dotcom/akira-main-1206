# Character Knowledge Index

## Назначение

Этот раздел фиксирует, что каждый персонаж знает, не знает и скрывает.

Файлы знаний не заменяют анкеты персонажей.

Анкета отвечает на вопрос: кто персонаж.

Knowledge-файл отвечает на вопрос: что персонаж знает на старте и что может или не может сказать.

## Правила

- Персонаж не может знать то, чего не видел, не слышал, не расследовал и о чём ему не сообщили.
- Знание персонажа не означает, что он обязан сказать это вслух.
- Скрытое знание нельзя превращать в прямой монолог без отдельной сцены.
- Если персонаж не знает другого персонажа на старте, он не должен узнавать его по имени, роли или цели.
- Внутренний канон используется для поведения, пауз, реакций и решений, а не для внезапных признаний.
- Если знание связано с кайрасами, использовать `canon/kairos_lore.md`.
- Если знание связано с происхождением Акиры, использовать `canon/akira_origin.md`.
- Если знание связано со школьным пожаром, использовать `canon/school_fire.md`.
